# Tim Harmar Legal & Consulting Services Website

A modern, professional website for Tim Harmar Legal & Consulting Services, built with React, Vite, and Tailwind CSS. The site features cutting-edge design, interactive elements, and is optimized for GitHub Pages deployment.

## 🚀 Features

- **Professional Design**: Modern, polished visual design with smooth animations
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Tools**: Smart scheduler, document generator, legal assessment tool
- **AI Legal Assistant**: Interactive chat widget for client engagement
- **SEO Optimized**: Meta tags, structured data, and social media optimization
- **Fast Performance**: Built with Vite for optimal loading speeds
- **Modern React**: Uses React 19 with hooks and modern patterns

## 🛠️ Technology Stack

- **Frontend**: React 19, JavaScript ES6+
- **Build Tool**: Vite 6.x
- **Styling**: Tailwind CSS 4.x
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **UI Components**: Radix UI primitives
- **Deployment**: GitHub Pages

## 📦 Installation

### Prerequisites
- Node.js 20+ 
- pnpm (preferred) or npm

### Setup
```bash
# Clone the repository
git clone https://github.com/harmartim-oss/THLAWSITE.git
cd THLAWSITE

# Install dependencies
pnpm install
# or
npm install --legacy-peer-deps

# Start development server
pnpm dev
# or
npm run dev

# Build for production
pnpm build
# or
npm run build

# Preview production build
pnpm preview
# or
npm run preview
```

## 🚀 Deployment

### GitHub Pages Deployment

The site is configured for automatic deployment to GitHub Pages:

1. **Automatic Deployment**: Push to `main` branch triggers automatic deployment
2. **Custom Domain**: Configured for `www.timharmar.com`
3. **Build Process**: Uses pnpm for faster, more reliable builds
4. **Production Optimization**: Minified CSS and JavaScript for optimal performance

### Manual Deployment

```bash
# Build the project
pnpm build

# The dist/ folder contains the deployable files
# Upload the contents of dist/ to your web server
```
